About

I'm William Stafford Parsons, a 33-year-old computer science researcher, game developer and web software engineer with an emphasis on minimalism.

I create tiny offline browser games that are fun to play for a few minutes at a time.

I conduct open computer science research to create improved fundamental algorithms and data structures.

I'm starting BitVanish, a collection of offline browser tools for compressing and converting encoded data with client-side privacy.

I'm starting LootConfirm, a two-way input system for randomizing digital in-game results with client-side fairness verifications to prevent cheating.

Furthermore, I aspire to contribute impactful enhancements to popular open-source projects.

I have no formal education beyond a year of community college. I'm jobless and unofficially disabled with a terminal-feeling illness.

I've been doing this work for free while believing the value will appreciate as humanity embraces technology on a deeper level.
